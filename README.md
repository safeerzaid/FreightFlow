# 🚛 FreightFlow

A logistics platform connecting customers with truck/tempo drivers 
for intercity freight in Kerala — solving the dead mile problem 
through backhaul matching.

## 🚀 Live Demo
Coming soon

## 🛠️ Tech Stack
- **Frontend:** React, TypeScript, Tailwind, Shadcn UI
- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Auth:** JWT, bcrypt
- **Payments:** Stripe
- **Real-time:** Socket.io
- **Deployment:** Vercel (frontend), Railway (backend)

## ⚙️ Setup
1. Clone the repo: `git clone https://github.com/yourusername/freightflow`
2. `cd backend && npm install`
3. Create `.env` file — see `.env.example` for required variables
4. `npm run dev`
